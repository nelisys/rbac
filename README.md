# Nelisys RBAC for Laravel

## Introduction

Role-Based Access Control for Laravel.

## Installation

Run composer to install the package.

```
composer require nelisys/rbac
```

```
php artisan vendor:publish --provider="Nelisys\Rbac\RbacServiceProvider"
```

```
php artisan migrate
```
